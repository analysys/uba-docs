# 项目概览

项目概览界面可以快速了解当前项目的基本情况

* 基础信息
  * 项目LOGO，移入LOGO可更新的
  * AppKey，用于集成SDK
  * 项目创建时间
* 埋点情况
  * 回数时间
  * 回数平台
  * 埋点事件数量
  * 埋点用户属性数量
  * 过去30日事件量
* 成员情况
  * 平台管理员，当需要在平台中添加新成员时，可以联系平台管理员
  * 项目管理员，当需要再项目中添加成员或者修改权限时，可联系项目管理员
  * 项目成员，项目管理员可以将平台成员添加到项目中，并设置不同的权限
